# Инструкция по инициализации бота zxcmod

## Шаг 1: Настройка загрузки эмоджи

Перед запуском бота необходимо загрузить все эмоджи в бота. Это позволит использовать до 2000 эмоджи без необходимости создавать сервера.

### 1.1. Изменение ID бота в emoji.py

1. Откройте файл `zxcmod/zxcmodcogs/emoji.py`
2. Найдите строку с `APPLICATION_ID` (строка 25):
   ```python
   APPLICATION_ID = "1405674113887895634"
   ```
3. Замените значение на ID вашего бота (Application ID можно найти в Discord Developer Portal)

### 1.2. Изменение цвета эмоджи (опционально)

Если нужно изменить цвет эмоджи под цветовую гамму вашего сервера:

1. Запустите бота
2. В канале, где бот имеет права, выполните команду:
   ```
   !change_color #HEX_КОД_ЦВЕТА
   ```
   Например: `!change_color #FF5733`
   
   Эта команда перекрасит все эмоджи в папке `images_upload` в указанный цвет.

### 1.3. Загрузка эмоджи в бота

После изменения цвета (или если цвет менять не нужно):

1. Убедитесь, что все файлы эмоджи находятся в папке `images_upload`
2. В канале выполните команду:
   ```
   !upload_emojis
   ```
   
   Бот автоматически:
   - Загрузит все эмоджи из папки `images_upload` в приложение бота
   - Сохранит информацию об эмоджи в базу данных
   - Можно загрузить до 2000 эмоджи

**Важно:** Команды `!change_color` и `!upload_emojis` доступны только пользователю с ID, указанным в `config['author']` (по умолчанию: 849353684249083914)

## Шаг 2: Настройка конфигурации

После загрузки эмоджи необходимо настроить конфигурационные файлы:

### 2.1. Настройка configs/zxc_tokens.json

Убедитесь, что в файле указаны правильные значения:
- `mongodb` - строка подключения к MongoDB
- `moderation` - токен бота для модерации

### 2.2. Настройка configs/zxc.json

Используйте команду `!config_set` (доступна только для автора) для автоматической настройки:
- Ролей
- Каналов
- Категорий
- Логов

Или настройте вручную, изменив значения в файле `configs/zxc.json`.

## Шаг 3: Запуск бота

После выполнения всех шагов:

1. Убедитесь, что MongoDB запущена и доступна
2. Запустите бота командой:
   ```bash
   python zxcmod.py
   ```
   или
   ```bash
   python zxcmod/zxcmod.py
   ```

## Важные замечания

- **Эмоджи загружаются в приложение бота**, а не на сервер Discord
- Это позволяет использовать до 2000 эмоджи без ограничений сервера
- Не нужно создавать сервера для хранения эмоджи
- Все эмоджи будут доступны боту на всех серверах, где он используется

## Порядок действий (кратко)

1. ✅ Изменить `APPLICATION_ID` в `zxcmod/zxcmodcogs/emoji.py`
2. ✅ (Опционально) Выполнить `!change_color #HEX` для изменения цвета
3. ✅ Выполнить `!upload_emojis` для загрузки эмоджи
4. ✅ Настроить `configs/zxc.json` (через `!config_set` или вручную)
5. ✅ Запустить бота: `python zxcmod.py`
